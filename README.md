# E.martBD Server

# For: EJP-NextJS-Task - Web - E.martBD Client

**E.martBD** is the backend server for the E.martBD application. It handles user authentication, product management, and secure API endpoints for the client. The server ensures smooth communication between the client and the database, providing features like user login, JWT-based session handling, and CRUD operations for products.

## Features

- **User Authentication:** Supports login with credentials and Google OAuth.
- **JWT Session Management:** Secure token-based authentication for protected routes.
- **Product Management:** APIs for adding, updating, deleting, and fetching products.
- **Secure Routing:** Middleware/proxy to restrict access to authenticated users.
- **Database Integration:** Connects to MongoDB for storing user and product data.
